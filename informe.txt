nasm -f elf main.asm -o main.o:

nasm: Es el comando para ensamblar código fuente en lenguaje ensamblador.
-f elf: Especifica el formato de salida como ELF (Executable and Linkable Format), que es el formato de archivo ejecutable comúnmente utilizado en sistemas operativos basados en UNIX.
main.asm: Es el nombre del archivo que contiene el código en ensamblador que deseas ensamblar.
-o main.o: Especifica el nombre del archivo de salida que contendrá el código ensamblado. En este caso, se crea un archivo llamado main.o.
Este comando ensambla el código en ensamblador del archivo main.asm y genera un archivo de objeto main.o.

gcc -m32 calculadora.o main.o -o calculadora:

gcc: Es el compilador GNU C.
-m32: Especifica que el código se compilará para una arquitectura de 32 bits.
calculadora.o main.o: Especifica los archivos de objeto que se deben enlazar para crear el ejecutable final. En este caso, calculadora.o contiene las funciones en ensamblador y main.o contiene la función principal escrita en ensamblador.
-o calculadora: Especifica el nombre del archivo ejecutable que se generará. En este caso, se crea un archivo ejecutable llamado calculadora.
Este comando enlaza los archivos de objeto para crear un ejecutable llamado calculadora.

gcc -m32 -c calculadora.c -o calculadora.o:

-c: Indica al compilador que solo compile el código fuente y genere un archivo de objeto, pero no lo enlace para crear un ejecutable final.
calculadora.c: Es el nombre del archivo que contiene el código en C que deseas compilar.
-o calculadora.o: Especifica el nombre del archivo de objeto que se generará. En este caso, se crea un archivo de objeto llamado calculadora.o.
Este comando compila el código en C del archivo calculadora.c y genera un archivo de objeto calculadora.o.

./calculadora:

./: Es el prefijo que indica que se está ejecutando un archivo en el directorio actual.
calculadora: Es el nombre del archivo ejecutable que se desea ejecutar.
Este comando ejecuta el archivo ejecutable calculadora.
